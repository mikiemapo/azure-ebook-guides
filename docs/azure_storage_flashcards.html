Storage Flashcards: Azure Storage Redundancy (AZ-104)

Tap/Click the blue header bar on each card below to expand or collapse the details.

Locally-redundant storage

Concept: Lowest cost. 3 synchronous copies within a single data center (physical building) in the primary region.

  • Copies: 3 (in one DC)
  • Scope: Single Data Center
  • Durability: 11 nines
  • Protects against: Drive/Rack failure.
  • Vulnerable to: DC-level disaster.
  • Read Access: Primary only.

Key Takeaway: LRS offers the lowest-cost redundancy, but only protects against hardware failure within a single data center—not site-wide disasters.

Zone-redundant storage

Concept: 3 synchronous copies across 3 different Availability Zones (AZs) within the primary region.

  • Copies: 3 (1 per AZ)
  • Scope: Multiple AZs (in 1 Region)
  • Durability: 12 nines
  • Protects against: DC/Zone failure.
  • Read Access: Primary only (any AZ).

Geo-redundant storage

Concept: LRS in primary (3 sync copies) + Asynchronous copy to LRS (3 sync copies) in a secondary region.

  • Copies: 3 (Pri) + 3 (Sec) = 6 Total
  • Scope: Two Regions
  • Durability: 16 nines
  • Protects against: Regional disaster.
  • Failover: Required to access secondary.
  • Read Access: Primary only (unless failed over).

Key Takeaways

  • LRS: Cheapest, protects against hardware failure within a single data center, but not against site-wide disasters.
  • ZRS: Protects against data center and zone failures by distributing data across 3 AZs in one region.
  • GRS: Adds regional disaster protection by asynchronously replicating LRS data to a paired region. Secondary is only accessible after failover.
  • RA-GRS: Like GRS, but enables read-only access to the secondary region for higher availability and global read performance.
  • GZRS: Combines ZRS (across AZs) with regional replication, giving both zone and regional failure protection. Secondary is only accessible after failover.
  • RA-GZRS: The most resilient option—zone and regional protection, plus global read access to the secondary region for maximum durability and availability.
